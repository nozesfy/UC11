# Modernização da Gestão de Estoque — Rede de Supermercados Atacadão

## Sobre o Projeto

Este repositório contém a documentação e os artefatos do projeto de modernização do processo de gestão de estoque de uma rede de supermercados.

## Problemas Identificados

- Planejamento de compras ineficiente
- Controle de estoque defasado
- Perdas não mapeadas
- Inventário manual e demorado
- Movimentação entre depósitos sem rastreabilidade
- Vendas (CPF e CNPJ) com processos distintos
- Falta de indicadores gerenciais

## Equipes

Quatro equipes de desenvolvimento atuam como consultorias de software, propondo soluções para os desafios listados.

## Estrutura

- `ATIVIDADE.md` — descrição detalhada do problema
- `CHANGELOG.md` — histórico de alterações do projeto
