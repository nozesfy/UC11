# Modernização da Gestão de Estoque — Rede de Supermercados Atacadão

## Contexto

Uma rede de supermercados pretende modernizar seu processo de gestão de estoque.

## Dificuldades Atuais

- Planejamento de compras
- Controle de estoque
- Perdas
- Inventário
- Movimentação entre depósitos
- Vendas para clientes CPF
- Vendas para clientes CNPJ
- Geração de indicadores

## Clientes

### Cliente CPF (Pessoa Física)
Consumidor final que realiza compras para uso próprio ou familiar. Busca preços competitivos e variedade de produtos.

### Cliente CNPJ (Pessoa Jurídica)
Empresas que compram para revenda (mercados, mercearias, restaurantes) ou para consumo institucional. Demandam volumes maiores, condições comerciais diferenciadas e emissão de notas fiscais adequadas.

## Proposta

A empresa contratou quatro equipes de desenvolvimento para propor soluções. Cada equipe atuará como uma consultoria de software.
