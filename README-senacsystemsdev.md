Somos uma equipe acadêmica do curso **Técnico em Desenvolvimento de Sistemas**, atuando no projeto **UC11 — Gerir Projetos de TI**.

## 🏪 Projeto: Gestão de Perdas — Atacadão

MVP de um sistema com inteligência artificial para **controle de validade, prevenção de perdas e detecção de anomalias** na rede de supermercados Atacadão.

🔗 [Acessar o projeto](https://senacsystemsdev.github.io/UC11/)

## 🛠️ Stack do Projeto

- **Frontend:** React 19, TypeScript, Tailwind CSS v4, Vite
- **Backend:** API REST (Node.js)
- **ML:** Modelos preditivos para detecção de anomalias e previsão de perdas
- **Banco de Dados:** PostgreSQL
- **Deploy:** GitHub Actions + GitHub Pages
